Dono do Zap — Validar Telefones

Consulta a API do Dono do Zap para verificar se cada número pertence ao sócio informado, a outro vínculo ou não tem registro.

O que faz

  1. Lê todas as colunas de telefone da planilha: Telefone, Telefone 2, Fone Movel Contato 1, Fone Movel Contato 2…
  2. Para cada número na célula (pode ter vários separados por linha), consulta a API do Dono do Zap e obtém os nomes vinculados ao WhatsApp.
  3. Compara o primeiro nome do retorno com o campo Nome contato N correspondente ao índice do telefone.
  4. Escreve o resultado na coluna VERIFICADO - N: nome real se bater com o sócio, Outros Vínculos se não bater, Sem dados se não houver registro na API.

Formatar Planilha

Aplica formatação visual ao arquivo Excel: cabeçalhos em azul escuro, bordas e fonte padronizada. Nenhum dado é alterado.

O que faz

  1. Lê todos os dados da planilha sem nenhuma verificação de colunas.
  2. Gera um arquivo .xlsx com cabeçalhos formatados (fundo azul escuro, texto branco, negrito), bordas finas em todas as células e fonte padronizada.
  3. Os dados são preservados exatamente como estão — nenhuma coluna é adicionada ou removida.

Colunas esperadas

Nenhuma. Este script funciona com qualquer planilha, independente dos cabeçalhos. Basta enviar o arquivo.

Avaliar ICP com IA

Usa IA para avaliar cada empresa contra a descrição do ICP informada no formulário. Atribui score de 0 a 100 e gera resumo e motivo.

O que faz

  1. Para cada linha, monta um prompt com Empresa, Cidade, Estado e Site.
  2. Envia para o provedor de IA configurado junto com a descrição do ICP preenchida no formulário.
  3. A IA retorna: site atualizado (se estava vazio), resumo da empresa em duas frases, score de 0–100 e motivo da avaliação em uma frase.
  4. Linhas sem nome de empresa são puladas automaticamente.
  5. Em caso de limite da API (erro 429), o script aguarda e tenta novamente com backoff progressivo.

Validar Tempo de Existência / MEI

Consulta a API pública da Receita Federal (BrasilAPI) por CNPJ. Sinaliza em vermelho empresas com menos de 1 ano de existência ou classificadas como MEI.

O que faz

  1. Lê o CNPJ de cada linha — formatação com pontos e traços é removida automaticamente.
  2. Consulta a BrasilAPI (brasilapi.com.br/api/cnpj/v1/…) para obter data de abertura e porte.
  3. Calcula a idade em dias. Se menor que 365, registra como alerta "menos de 365 dias".
  4. Detecta MEI/SIMEI pelo campo de porte ou natureza jurídica retornado pela Receita.
  5. Linhas com qualquer alerta são pintadas de vermelho no arquivo Excel de saída.

Unificar Empresas + Sócios

Cruza duas planilhas (empresas e sócios) pelo CNPJ e gera uma planilha unificada com os dados de contato de cada sócio por empresa.

O que faz

  1. Lê a planilha de empresas e a planilha de sócios separadamente.
  2. Agrupa os sócios por CNPJ e os ordena pelo campo prioridade (menor número = primeiro na lista).
  3. Para cada empresa, busca todos os sócios com o mesmo CNPJ.
  4. Gera uma linha por empresa com os dados de cada sócio numerados: Nome contato 1, Nome contato 2… até quantos sócios houver.
  5. O resultado final é ordenado por empresas com menos sócios primeiro.

Zenvia — Adicionar DDI 55

Padroniza todos os números de telefone da planilha para o formato da API Zenvia: somente dígitos com o DDI 55 do Brasil na frente.

O que faz

  1. Percorre todas as colunas cujo nome seja reconhecido como telefone (ver lista abaixo).
  2. Divide o conteúdo de cada célula por quebra de linha, /, ;, , ou | — detecta múltiplos números por célula.
  3. Para cada número: remove toda formatação (parênteses, traços, espaços) e adiciona 55 na frente.
  4. Se o número já começa com 55, mantém como está — não duplica o DDI.
  5. Múltiplos números ficam separados por quebra de linha no resultado.

Colunas detectadas automaticamente por nome

Não há colunas obrigatórias. O script identifica colunas de telefone pelas palavras-chave abaixo no cabeçalho:
